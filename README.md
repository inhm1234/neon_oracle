# 오늘의 운세코드

AI 운세 분석 웹사이트 개발 버전입니다.

## 현재 버전

V3-3

## 핵심 기능

- 생년월일 / 태어난 시간 기반 오늘의 운세 분석
- 루미 / 모코 / 노바 파트너 시스템
- 파트너 직접 선택 / 랜덤 뽑기 / 교체
- 운세 결과에 따른 파트너 해석
- EXP / 레벨업 / 성장 단계
- 파트너 감정 반응 / 표정 분위기 변화
- 출석 보상 / 연속 출석 / 보상 기록
- 파트너 도감 / 프로필 화면
- 이전 운세 다시보기
- 데이터 관리 센터
- 백업 파일 다운로드 / 백업 파일 불러오기
- 저장 데이터 전체 초기화
- 프로필별 저장 구조
- 프로필 이름 변경 / 복사 / 삭제
- 로그인 저장 구조 점검
- Firebase Google 로그인 / 로그아웃
- Firestore 수동 서버 저장 / 서버 불러오기
- 서버 / 브라우저 데이터 비교
- 로그인 후 서버 상태 자동 확인
- 개발 점검 체크리스트
- 체크리스트 상태 새로고침 후 유지

## V3-3 추가 기능

- 로그인 후 서버 상태 자동 확인 카드 추가
- Google 로그인 상태가 유지되어 있으면 페이지 진입 시 서버 상태 자동 확인
- 로그인 직후 Firestore 서버 데이터 존재 여부 자동 확인
- 자동 확인 결과: 서버 데이터 없음 / 거의 동일 / 서버가 더 최신일 수 있음 / 브라우저가 더 최신일 수 있음 / 차이 있음 표시
- 마지막 자동 확인 시간 표시
- 자동 확인 다시 실행 버튼 추가
- 자동 저장 / 자동 불러오기는 아직 하지 않음
- V3-2 서버/브라우저 비교 안전장치 유지
- V3-1 수동 서버 저장 / 불러오기 유지
- V3-0.1 Google 로그인 / 로그아웃 유지
