# 오늘의 운세코드

AI 운세 분석 웹사이트 개발 버전입니다.

## 현재 버전

V3-12

## 이전 안정 버전

V3-11.1 stable

## 핵심 기능

- 생년월일 / 태어난 시간 기반 오늘의 운세 분석
- Google 로그인 / 로그아웃
- 카카오톡 등 앱 내부 브라우저 접속 안내
- 모바일 리다이렉트 로그인
- Firestore 서버 저장 / 불러오기
- 관리자 전용 통계 대시보드
- 일반 사용자 화면과 관리자 도구 분리
- 루미 / 모코 / 노바 파트너 시스템
- 출석 보상 / 연속 출석 / 보상 기록
- 파트너 도감 / 프로필 화면
- 이전 운세 다시보기
- 내 데이터 관리 / 백업 파일 다운로드 / 백업 파일 불러오기

## V3-12 추가 기능

- 검색 유입 준비용 SEO 기본 세팅 추가
- 검색 결과에 어울리는 페이지 제목과 설명문 추가
- canonical URL / Open Graph / Twitter Card 메타 태그 추가
- WebApplication 구조화 데이터(JSON-LD) 추가
- 검색엔진 안내용 sitemap.xml 추가
- 검색로봇 기본 안내용 robots.txt 추가
- 일반 화면 아래쪽에 검색용 서비스 소개 영역 추가
- manifest.json에 앱 설명 추가
- V3-11.1의 일반 사용자 / 관리자 화면 분리 구조 유지

## V3-11.1 안정 확인 내용

- 일반 주소에서는 관리자 계정으로 로그인되어 있어도 관리자 도구가 보이지 않음
- `?admin=1` 주소와 관리자 Google 계정에서만 관리자 도구 확인 가능
- 관리자 통계 대시보드 표시 확인
- 통계 새로고침 및 운세 분석 클릭 수 증가 확인

## 적용 방법

GitHub Pages 저장소에 아래 파일을 덮어씌웁니다.

- index.html
- style.css
- app.js
- manifest.json
- service_worker.js
- README.md
- CHANGELOG.md
- VERSION.txt
- robots.txt
- sitemap.xml
- icon-192.png
- icon-512.png

업로드 후 Ctrl + F5로 강력 새로고침합니다.

## 확인 체크리스트

- 일반 주소에서 첫 화면이 심플하게 유지되는지 확인
- 운세 분석 버튼이 정상 작동하는지 확인
- 일반 주소에서 관리자 도구가 보이지 않는지 확인
- `?admin=1` 주소에서 관리자 도구와 통계 대시보드가 보이는지 확인
- 브라우저 탭 제목이 검색용 문구로 바뀌었는지 확인
- 화면 아래쪽에 오늘의 운세코드 소개 영역이 보이는지 확인
- `/robots.txt` 주소가 열리는지 확인
- `/sitemap.xml` 주소가 열리는지 확인
