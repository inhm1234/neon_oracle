# 오늘의 운세코드

AI 운세 분석 웹사이트 개발 버전입니다.

## 현재 버전

V3-6

## 핵심 기능

- 생년월일 / 태어난 시간 기반 오늘의 운세 분석
- 루미 / 모코 / 노바 파트너 시스템
- 파트너 직접 선택 / 랜덤 뽑기 / 교체
- 운세 결과에 따른 파트너 해석
- EXP / 레벨업 / 성장 단계
- 파트너 감정 반응 / 표정 분위기 변화
- 출석 보상 / 연속 출석 / 보상 기록
- 파트너 도감 / 프로필 화면
- 이전 운세 다시보기
- 데이터 관리 센터
- 백업 파일 다운로드 / 백업 파일 불러오기
- 저장 데이터 전체 초기화
- 프로필별 저장 구조
- 프로필 이름 변경 / 복사 / 삭제
- 로그인 저장 구조 점검
- Firebase Google 로그인 / 로그아웃
- Firestore 수동 서버 저장 / 서버 불러오기
- 서버 / 브라우저 데이터 비교
- 로그인 후 서버 상태 자동 확인
- 선택형 자동 동기화 준비
- 동기화 선택값 저장 / 설정 유지
- 브라우저 우선 / 서버 우선 추천 실행 강화
- 개발 점검 체크리스트
- 체크리스트 상태 새로고침 후 유지

## V3-6 추가 기능

- 브라우저/서버 우선 추천 실행 강화 카드로 개선
- 선택한 동기화 방식에 맞춰 추천 실행 영역 표시
- 브라우저 우선 선택 시 서버 저장을 우선 추천
- 서버 우선 선택 시 서버 불러오기를 우선 추천
- 서버 데이터가 없을 때는 첫 서버 저장을 추천
- 추천 실행 버튼 추가
- 추천 실행 버튼도 기존 저장/불러오기 확인창을 거침
- 자동 저장 / 자동 불러오기는 아직 하지 않음
- V3-5 선택값 저장 및 설정 유지 기능 유지
- V3-4 선택형 자동 동기화 준비 기능 유지
- V3-3 로그인 후 서버 상태 자동 확인 유지
- V3-2 서버/브라우저 비교 안전장치 유지
- V3-1 수동 서버 저장 / 불러오기 유지
- V3-0.1 Google 로그인 / 로그아웃 유지
