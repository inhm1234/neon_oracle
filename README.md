# 오늘의 운세코드

AI 운세 분석 웹사이트 개발 버전입니다.

## 현재 버전

V3-2

## 핵심 기능

- 생년월일 / 태어난 시간 기반 오늘의 운세 분석
- 루미 / 모코 / 노바 파트너 시스템
- 파트너 직접 선택 / 랜덤 뽑기 / 교체
- 운세 결과에 따른 파트너 해석
- EXP / 레벨업 / 성장 단계
- 파트너 감정 반응 / 표정 분위기 변화
- 출석 보상 / 연속 출석 / 보상 기록
- 파트너 도감 / 프로필 화면
- 이전 운세 다시보기
- 데이터 관리 센터
- 백업 파일 다운로드 / 백업 파일 불러오기
- 저장 데이터 전체 초기화
- 프로필별 저장 구조
- 프로필 이름 변경 / 복사 / 삭제
- 로그인 저장 구조 점검
- 서버 / 브라우저 데이터 비교
- 로그인 저장용 데이터 미리보기
- Firebase Google 로그인 / 로그아웃
- 로그인 사용자 이름 / 이메일 표시
- Firestore 로그인 계정 수동 저장 / 불러오기 테스트
- 개발 점검 체크리스트
- 체크리스트 상태 새로고침 후 유지

- 서버 데이터 없음 / 첫 저장 필요 / 차이 있음 / 거의 동일 상태 안내

## V3-2 확인 방법

- 화면에 V3-2이 보이는지 확인합니다.
- Firebase 로그인 연결 테스트 카드가 보이는지 확인합니다.
- Google 로그인 / 로그아웃이 정상 작동하는지 확인합니다.
- 로그인 계정 저장 테스트 카드가 보이는지 확인합니다.
- 서버/브라우저 비교 버튼이 작동하는지 확인합니다.
- 현재 데이터를 서버에 저장할 때 확인창이 뜨는지 확인합니다.
- 서버 상태 확인이 작동하는지 확인합니다.
- 서버에서 데이터를 다시 불러올 때 경고창이 뜨는지 확인합니다.
- 기존 프로필 / 백업 / 운세 기록 / 출석 보상이 그대로 작동하는지 확인합니다.
