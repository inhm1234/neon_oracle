# 오늘의 운세코드

AI 운세 분석 웹사이트 개발 버전입니다.

## 현재 버전

V3-8.1

## 핵심 기능

- 생년월일 / 태어난 시간 기반 오늘의 운세 분석
- Google 로그인 / 로그아웃
- 카카오톡 등 앱 내부 브라우저 접속 안내
- 모바일 리다이렉트 로그인
- Firestore 서버 저장 / 불러오기
- 서버 / 브라우저 데이터 비교
- 로그인 후 서버 상태 자동 확인
- 동기화 선택 방식 저장 유지
- 추천 실행 기능
- 루미 / 모코 / 노바 파트너 시스템
- 파트너 직접 선택 / 랜덤 뽑기 / 교체
- 운세 결과에 따른 파트너 해석
- EXP / 레벨업 / 성장 단계
- 파트너 감정 반응 / 표정 분위기 변화
- 출석 보상 / 연속 출석 / 보상 기록
- 파트너 도감 / 프로필 화면
- 이전 운세 다시보기
- 데이터 관리 센터
- 백업 파일 다운로드 / 백업 파일 불러오기
- 프로필별 저장 구조
- 개발 점검 체크리스트

## V3-8.1 추가 기능

- 카카오톡 / 인스타그램 / 네이버앱 등 앱 내부 브라우저 접속 감지
- 앱 내부 브라우저에서 Google 로그인 오류 화면으로 바로 보내지 않고 안내 표시
- 운세 먼저 보기 / 브라우저로 열기 / 주소 복사 버튼 추가
- 모바일 일반 브라우저에서는 리다이렉트 방식으로 Google 로그인 진행
- PC에서는 기존 팝업 Google 로그인 유지
- 기존 V3-8 심플 홈 구조 유지

## V3-8 추가 기능

- 첫 화면을 운세 입력 중심으로 단순화
- Google 계정 저장 영역을 작고 간단한 상태 표시 형태로 정리
- 파트너 / 출석 / 이전 운세 / 파트너 도감을 기본 접힘 상태로 변경
- 파트너 도감은 사용자가 궁금할 때 직접 열어보는 서브 콘텐츠로 변경
- 고급 설정 / 개발자 도구는 기존처럼 접힘 상태 유지
- 기존 V3-6 서버 저장 / 불러오기 / 비교 / 추천 실행 기능 유지
- 기존 Google 로그인 / 로그아웃 유지
- 기존 운세 분석 / 파트너 / 출석 / 백업 기능 유지
