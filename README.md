# 오늘의 운세코드

AI 운세 분석 웹사이트 개발 버전입니다.

## 현재 버전

V2-3.3

## 핵심 기능

- 생년월일 / 태어난 시간 기반 오늘의 운세 분석
- 루미 / 모코 / 노바 파트너 시스템
- 파트너 직접 선택 / 랜덤 뽑기 / 교체
- 운세 결과에 따른 파트너 해석
- EXP / 레벨업 / 성장 단계
- 개발 점검 체크리스트
- 체크리스트 상태 새로고침 후 유지

## 적용 방법

GitHub Pages 저장소에 아래 파일을 덮어씌웁니다.

- index.html
- style.css
- app.js
- manifest.json
- service_worker.js

업로드 후 Ctrl + F5로 강력 새로고침합니다.
