# 오늘의 운세코드

AI 운세 분석 웹사이트 개발 버전입니다.

## 현재 버전

V3-4

## 핵심 기능

- 생년월일 / 태어난 시간 기반 오늘의 운세 분석
- 루미 / 모코 / 노바 파트너 시스템
- 파트너 직접 선택 / 랜덤 뽑기 / 교체
- 운세 결과에 따른 파트너 해석
- EXP / 레벨업 / 성장 단계
- 파트너 감정 반응 / 표정 분위기 변화
- 출석 보상 / 연속 출석 / 보상 기록
- 파트너 도감 / 프로필 화면
- 이전 운세 다시보기
- 데이터 관리 센터
- 백업 파일 다운로드 / 백업 파일 불러오기
- 저장 데이터 전체 초기화
- 프로필별 저장 구조
- 프로필 이름 변경 / 복사 / 삭제
- 로그인 저장 구조 점검
- Firebase Google 로그인 / 로그아웃
- Firestore 수동 서버 저장 / 서버 불러오기
- 서버 / 브라우저 데이터 비교
- 로그인 후 서버 상태 자동 확인
- 선택형 자동 동기화 준비
- 개발 점검 체크리스트
- 체크리스트 상태 새로고침 후 유지

## V3-4 추가 기능

- 선택형 자동 동기화 준비 카드 추가
- 동기화 선택 방식 저장 추가: 항상 수동 / 추천만 보기 / 브라우저 우선 / 서버 우선
- 서버 상태 확인 후 저장 또는 불러오기 추천 문구 표시 추가
- 추천 기준으로 서버 저장 / 서버 불러오기 버튼 연결
- 실제 저장 / 불러오기는 기존 확인창을 거쳐 사용자가 직접 실행
- 자동 저장 / 자동 불러오기는 아직 하지 않음
- V3-3 로그인 후 서버 상태 자동 확인 유지
- V3-2 서버/브라우저 비교 안전장치 유지
- V3-1 수동 서버 저장 / 불러오기 유지
- V3-0.1 Google 로그인 / 로그아웃 유지
