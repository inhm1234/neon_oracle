# 오늘의 운세코드

AI가 생년월일과 태어난 시간을 바탕으로 오늘의 운세를 분석해주는 GitHub Pages용 웹사이트입니다.

## 현재 버전

V2-3.1

## 이번 버전 핵심

- V2-3의 파트너 직접 선택 / 랜덤 뽑기 / 교체 기능 유지
- 사이트 안에 개발 점검 체크리스트 추가
- 체크 상태를 localStorage에 저장
- 화면 하단에 현재 버전 표시

## 적용 파일

아래 파일을 GitHub 저장소에 그대로 덮어씌우면 됩니다.

- index.html
- style.css
- app.js
- manifest.json
- service_worker.js

## 적용 후 확인

사이트를 연 뒤 아래를 확인하세요.

1. 화면 아래에 `오늘의 운세코드 V2-3.1` 표시가 보이는지 확인
2. `업데이트 점검표` 항목을 하나씩 눌러 직접 체크
3. 새로고침 후 체크 상태가 유지되는지 확인

## 주의

이 버전은 개발 점검용 체크리스트가 포함된 버전입니다. 체크리스트는 테스트를 돕는 기능이며, 사용자가 직접 확인 후 체크하는 방식입니다.
