# 오늘의 운세코드

AI 운세 분석 웹사이트 개발 버전입니다.

## 현재 버전

V3-7

## 핵심 기능

- 생년월일 / 태어난 시간 기반 오늘의 운세 분석
- 루미 / 모코 / 노바 파트너 시스템
- 운세 결과에 따른 파트너 해석
- EXP / 레벨업 / 성장 단계
- 출석 보상 / 연속 출석 / 보상 기록
- 파트너 도감 / 프로필 화면
- 이전 운세 다시보기
- Google 로그인 / 로그아웃
- Firestore 서버 저장 / 불러오기
- 서버 / 브라우저 데이터 비교
- 동기화 설정 유지
- 데이터 백업 / 복원
- 프로필 관리
- 개발 점검 체크리스트

## V3-7 추가 기능

- 기본 화면을 사용자 중심으로 정리
- 운세 입력 영역을 화면 상단 쪽으로 이동
- Google 계정 저장 카드를 간단한 상태 표시 형태로 정리
- 서버 저장 / 서버 불러오기 / 비교 / 동기화 설정을 고급 설정 안으로 이동
- 데이터 관리 센터 / 로그인 저장 구조 점검 / 개발 점검표를 고급 설정 안으로 이동
- 일반 사용자가 운세 보기 흐름에 집중할 수 있도록 화면 복잡도 감소
- V3-6 추천 실행 기능 유지
- V3-5 동기화 설정 저장 유지
- V3-3 서버 상태 자동 확인 유지
- V3-1 서버 저장 / 불러오기 유지
