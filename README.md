# 오늘의 운세코드

AI 운세 분석 웹사이트 개발 버전입니다.

## 현재 버전

V2-9

## 핵심 기능

- 생년월일 / 태어난 시간 기반 오늘의 운세 분석
- 루미 / 모코 / 노바 파트너 시스템
- 파트너 직접 선택 / 랜덤 뽑기 / 교체
- 운세 결과에 따른 파트너 해석
- EXP / 레벨업 / 성장 단계
- 파트너 감정 반응 / 표정 분위기 변화
- 출석 보상 / 연속 출석 / 보상 기록
- 파트너 도감 / 프로필 화면
- 이전 운세 다시보기
- 데이터 관리 센터
- 백업 파일 다운로드 / 백업 파일 불러오기
- 저장 데이터 전체 초기화
- 개발 점검 체크리스트
- 체크리스트 상태 새로고침 후 유지
