# 오늘의 운세코드

AI 운세 분석 웹사이트 개발 버전입니다.

## 현재 버전

V3-0

## 핵심 기능

- 생년월일 / 태어난 시간 기반 오늘의 운세 분석
- 루미 / 모코 / 노바 파트너 시스템
- 파트너 직접 선택 / 랜덤 뽑기 / 교체
- 운세 결과에 따른 파트너 해석
- EXP / 레벨업 / 성장 단계
- 파트너 감정 반응 / 표정 분위기 변화
- 출석 보상 / 연속 출석 / 보상 기록
- 파트너 도감 / 프로필 화면
- 이전 운세 다시보기
- 데이터 관리 센터
- 백업 파일 다운로드 / 백업 파일 불러오기
- 저장 데이터 전체 초기화
- 프로필별 저장 구조
- 프로필 이름 변경 / 복사 / 삭제
- 로그인 저장 구조 점검
- 로그인 저장용 데이터 미리보기
- Firebase 로그인 연결 테스트
- Google 로그인 / 로그아웃 테스트
- 로그인 사용자 이름 / 이메일 표시
- 개발 점검 체크리스트
- 체크리스트 상태 새로고침 후 유지

## V3-0 추가 기능

- Firebase 연결 테스트 카드 추가
- Firebase SDK 동적 로드 구조 추가
- Google 로그인 테스트 버튼 추가
- 로그아웃 버튼 추가
- 로그인한 사용자 이름 / 이메일 / 프로필 이미지 표시
- Firebase 프로젝트 / 승인 도메인 / Google 제공업체 / Firestore 준비 상태 표시
- Firestore 데이터 저장은 아직 하지 않고, 로그인 연결만 확인
- V2-11 로그인 저장 구조 점검 유지
- V2-10 프로필 이름 변경 / 복사 / 삭제 유지
- V2-9 프로필별 저장 구조 유지
- V2-8 백업 / 복원 유지
- V2-7 출석 보상 유지

## 적용 방법

GitHub Pages 저장소에 아래 파일을 덮어씌웁니다.

- index.html
- style.css
- app.js
- manifest.json
- service_worker.js
- README.md
- CHANGELOG.md
- VERSION.txt
- icon-192.png
- icon-512.png

업로드 후 Ctrl + F5로 강력 새로고침합니다.

## V3-0 확인 방법

- Firebase 로그인 연결 테스트 카드가 보이는지 확인합니다.
- Firebase 상태가 준비 완료로 바뀌는지 확인합니다.
- Google 로그인 테스트 버튼을 누릅니다.
- 로그인 후 이름과 이메일이 표시되는지 확인합니다.
- 로그아웃 버튼이 작동하는지 확인합니다.
- 기존 프로필 / 백업 / 운세 기록 / 출석 보상이 그대로 작동하는지 확인합니다.
