## V3-10 관리자 통계 사용 전 주의

V3-10은 정적 GitHub Pages 환경에서 Firestore에 숫자 카운터를 저장하는 테스트 버전입니다.
관리자 화면은 기본 화면에서 숨김 상태입니다. 관리자 설정은 주소 뒤에 `?admin=1`을 붙여 들어간 뒤 Google 로그인 후 현재 계정을 등록하는 방식입니다.
실제 보안을 강화하려면 Firebase Console에서 Firestore 보안 규칙도 함께 설정하는 것을 권장합니다.
방문 수는 브라우저 기준 1일 1회로 기록되며, 실제 사람 수와 완전히 일치하지는 않을 수 있습니다.
